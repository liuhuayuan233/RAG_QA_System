# 机器学习算法详解

## 线性回归（Linear Regression）

线性回归是最基本的机器学习算法之一，用于预测连续数值。

### 基本原理

线性回归假设目标变量与特征之间存在线性关系：
y = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ + ε

其中：
- y：目标变量
- x₁, x₂, ..., xₙ：特征变量
- β₀, β₁, ..., βₙ：回归系数
- ε：误差项

### 损失函数

线性回归通常使用最小二乘法（MSE）作为损失函数：
MSE = (1/n) * Σ(yᵢ - ŷᵢ)²

### 优化方法

1. 正规方程法：直接求解解析解
2. 梯度下降法：迭代优化方法

## 逻辑回归（Logistic Regression）

逻辑回归用于二分类问题，使用sigmoid函数将线性组合映射到概率。

### Sigmoid函数

σ(z) = 1 / (1 + e^(-z))

### 损失函数

逻辑回归使用交叉熵损失：
Loss = -[y*log(p) + (1-y)*log(1-p)]

## 决策树（Decision Tree）

决策树通过递归分割特征空间来构建预测模型。

### 分割准则

1. 信息增益（Information Gain）
2. 基尼不纯度（Gini Impurity）
3. 均方误差（MSE）

### 剪枝技术

- 预剪枝：在构建过程中提前停止
- 后剪枝：构建完成后删除部分节点

## 随机森林（Random Forest）

随机森林是基于决策树的集成学习方法。

### 核心思想

1. Bagging：自助采样生成多个训练集
2. 随机特征选择：每次分割时随机选择特征子集
3. 投票机制：多个决策树的结果投票

### 优势

- 减少过拟合
- 处理缺失值
- 特征重要性评估

## 支持向量机（SVM）

SVM通过寻找最优超平面来分类数据。

### 核心概念

- 支持向量：距离分类超平面最近的样本点
- 间隔：两个类别之间的最大距离
- 核函数：将数据映射到高维空间

### 常用核函数

1. 线性核：K(x,y) = x·y
2. 多项式核：K(x,y) = (x·y + c)^d
3. RBF核：K(x,y) = exp(-γ||x-y||²)

## K-means聚类

K-means是最常用的聚类算法之一。

### 算法步骤

1. 随机初始化k个聚类中心
2. 将每个样本分配到最近的聚类中心
3. 更新聚类中心为所属样本的均值
4. 重复步骤2-3直到收敛

### 优化目标

最小化类内平方和（WCSS）：
WCSS = Σ Σ ||x - μᵢ||²

## 神经网络（Neural Network）

### 基本结构

- 输入层：接收输入特征
- 隐藏层：特征变换和非线性映射
- 输出层：产生最终预测

### 激活函数

1. Sigmoid：σ(x) = 1/(1+e^(-x))
2. Tanh：tanh(x) = (e^x - e^(-x))/(e^x + e^(-x))
3. ReLU：f(x) = max(0, x)
4. Leaky ReLU：f(x) = max(αx, x)

### 反向传播

通过链式法则计算梯度，更新网络参数。

## 集成学习

### Bagging

- 并行训练多个模型
- 通过平均或投票结合预测
- 代表算法：随机森林

### Boosting

- 串行训练多个弱学习器
- 后续模型关注前一个模型的错误
- 代表算法：AdaBoost、Gradient Boosting、XGBoost

### Stacking

- 使用元学习器结合多个基学习器
- 两层结构：基学习器 + 元学习器

## 模型评估

### 分类指标

- 准确率（Accuracy）
- 精确率（Precision）
- 召回率（Recall）
- F1分数
- ROC曲线和AUC

### 回归指标

- 均方误差（MSE）
- 平均绝对误差（MAE）
- 决定系数（R²）
- 均方根误差（RMSE）

### 交叉验证

- K折交叉验证
- 留一法（LOO）
- 时间序列交叉验证

## 特征工程

### 特征选择

- 过滤法：基于统计指标
- 包装法：基于模型性能
- 嵌入法：基于模型内部机制

### 特征变换

- 标准化：z = (x - μ)/σ
- 归一化：x' = (x - min)/(max - min)
- 对数变换：log(x)
- 多项式特征：x², x³, x₁x₂等

## 降维技术

### 主成分分析（PCA）

- 寻找数据的主要变化方向
- 保留最大方差的主成分
- 线性降维方法

### t-SNE

- 非线性降维技术
- 保持数据的局部结构
- 适用于可视化

## 推荐系统

### 协同过滤

- 基于用户的协同过滤
- 基于物品的协同过滤
- 矩阵分解技术

### 内容推荐

- 基于物品特征的推荐
- TF-IDF和余弦相似度
- 深度学习推荐

## 时间序列分析

### 传统方法

- ARIMA模型
- 季节性分解
- 指数平滑

### 深度学习方法

- LSTM和GRU
- 注意力机制
- Transformer用于时间序列
